# Udemy Data Analysis Project

This project involves analyzing Udemy course data using Python's data analysis and visualization libraries like **pandas**, **matplotlib**, and **seaborn**.

## 📊 Objective

To explore the Udemy course dataset and gain insights into:
- Course pricing patterns
- Popular categories and subcategories
- Free vs Paid courses
- Ratings and Reviews
- Instructor contributions

## 🧰 Tools and Libraries Used

- Python
- pandas
- matplotlib
- seaborn
- Jupyter Notebook (or any Python IDE)

## 📁 Files

- `udemy_data.csv` – The dataset used
- `analysis.ipynb` – Jupyter notebook with all the code and visualizations
- `README.md` – Project documentation

## 📈 Key Insights

- Paid courses dominate the platform, but free courses have high enrollments.
- Certain categories like "Web Development" and "Business" are very popular.
- Price range varies significantly across subcategories
